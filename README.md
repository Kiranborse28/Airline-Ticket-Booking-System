# ✈️ Airline Ticket Booking System

## 📖 Introduction

The Airline Ticket Booking System is a web-based application developed using PHP and MySQL that helps users book airline tickets online easily and efficiently. The system provides a user-friendly interface for searching flights, booking tickets, and managing reservations. It also includes an admin panel for managing flights, users, and booking records.

This project was developed as an academic project to understand real-world web application development concepts including database connectivity, authentication, CRUD operations, and backend management.

2. User registers or logs into the system
3. User searches available flights
4. Flight details are displayed
5. User books tickets
6. Booking details are stored in the database
7. Admin manages flights and booking records
8. User can view reservation information

# ⚙️ Detailed Process

Users create an account or log into the system securely.

# 📌 Guidelines

* Install XAMPP before running the project
* Start Apache and MySQL services
* Import the database into phpMyAdmin
* Place the project folder inside:
  C:\xampp\htdocs\
* Run the project using:
  http://localhost/airline_ticket_booking
